Resents — E-Commerce Fashion Website

Resents is a modern, fully responsive front-end e-commerce fashion website designed for trend-focused users who want a seamless and visually engaging online shopping experience. Built using HTML5, CSS3, and vanilla JavaScript, this project demonstrates how powerful and professional web applications can be created without relying on external frameworks.

The platform features a sleek dark-themed UI with gold accents, inspired by premium fashion brands while remaining affordable and accessible for students and young audiences.
